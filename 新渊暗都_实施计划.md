## 一、世界观设定

### 1.1 项目属性

| 项目 | 内容 |
|------|------|
| 项目名 | `新渊暗都` |
| 世界书名 | `新渊暗都` |
| 形式 | 角色卡（charactercard，内嵌世界书） |
| MVU | ✅ 启用 |
| EJS | ✅ 启用（区域/时间/角色阶段动态加载） |
| 头像 | 无 → JSON 输出 |
| 风格 | 现代都市 + 黑暗 + 多种族共存 |

### 1.2 世界概要

2026 年的超级都市「新渊城」——多种族共存的垂直城市。表面上霓虹璀璨、科技繁荣，暗地里帮派割据、异能者猎杀、深渊议会在阴影中编织阴谋。三类强化路线（灵能觉醒 / 基因飞升 / 机械义体）撕裂了社会契约。

### 1.3 种族体系（三大类·八种族）

#### 分类框架

| 一级分类 | 含义 | 包含种族 |
|---------|------|---------|
| **人类种** | 纯血 Homo sapiens | 人类 |
| **亚人种** | 人型变异/古代血脉/异界遗民 | 精灵种 / 兽化种 / 血族种 / 天使种 / 恶魔种 |
| **机械种** | 义体改造/机械意识 | 半机械体 / 纯机械生命体 |

#### 完整种族表

| # | 种族 | 分类 | 稀有度 | 性别锁 | 势力限制 | 核心特征 |
|---|------|------|--------|--------|---------|---------|
| 1 | 人类 | 人类种 | 普通 | 无 | 无 | 适应性极强，三条强化路线均无先天偏向 |
| 2 | 精灵种 | 亚人种 | 普通 | 无 | 无 | 远古自然契约后裔，寿命300+，基因强化上限+10 |
| 3 | 兽化种 | 亚人种 | 普通 | 无 | 无 | 兽灵血统/基因突变，感官敏锐，灵能初始+10 |

| 5 | 半机械体 | 机械种 | 特殊 | 无 | 不可深渊议会 | 先天义体化20%，义体上限+15，灵能亲和-10 |
| 6 | 天使种 | 亚人种 | 传说 | 仅女 | 仅城防署+自由行者 | 圣光审判庭遗民，光系灵能锁定，自带圣光翼 |

#### 亚人种来源 Lore

| 亚人分支 | 来源 | 社会地位 |
|---------|------|---------|
| 精灵种 | 远古人类与「自然之灵」订立契约的氏族后裔。血脉稳定，可遗传。 | 上城区古老家族，多为商会/学术精英 |
| 兽化种 | 基因突变或远古兽灵血统隔代觉醒。随机出现，不限于血统。 | 遍布各阶层，下层劳动者和街头帮派中占比高 |
| 血族种 | 被「初源之血」转化的女性血脉。无法通过男性传承，每位血族终其一生仅可转化一名继承人。 | 隐藏于夜色，多活跃于下城区和情报暗网 |
| 天使种 | 传说中「圣光审判庭」的唯一幸存血脉。审判庭覆灭后仅余女性后代，多数人认为已灭绝。 | 极稀少，通常隐匿身份生活在城防署或自由行者中 |
| 恶魔种 | 深渊裂隙大崩坏时，选择叛逃至人类一方的恶魔后裔。血脉自带「地狱火」印记，与深渊议会有复杂历史渊源。 | 被社会恐惧排斥，多在自由行者或暮光商会中谋生 |

## 二、势力与职业

### 2.1 四方势力

| 势力 | 定位 | 特点 | 可选职业 | 总部 |
|------|------|------|---------|------|
| **城防署（CDA）** | 官方治安 | 维护表面秩序，内部腐败与理想主义并存 | 外勤特工 / 灵能调查员 | 上城区·城防署总部 |
| **深渊议会** | 绝对反派 | 试图融合三路线创造「神之容器」，渗透全城 | 暗影执行者 / 议会学者 | 深渊裂隙 |
| **自由行者联盟** | 零散·佣兵 | 赏金猎人、雇佣兵、街头医生，只认钱不认人 | 赏金猎人 / 街区医生 | 中城区·自由行者酒吧 |
| **暮光商会** | 零散·中立 | 情报贩子、器械师、黑市商人，一切皆可交易 | 情报掮客 / 器械师 | 上城区·暮光交易所 |

### 2.2 力量体系（三条强化路线）

| 路线 | 名称 | 获取方式 | 风险 | 备注 |
|------|------|---------|------|------|
| 灵能飞升 | 灵能觉醒 | 自然觉醒/药物诱发 | 觉醒时可能死亡，女性存活率显著更高 | 纯机械生命体不可用 |
| 基因飞升 | 基因强化 | 注射/手术 | 排斥反应风险 | 纯机械生命体不可用 |
| 机械飞升 | 义体改造 | 手术替换 | 赛博精神病风险 | 半机械体/纯机械生命体先天拥有 |

## 六、面板设计

### 6.1 角色创建面板（`正则/角色创建面板.html`）

**流程编排（6步）**：

```
STEP 1: 选择势力 ──→ 预览势力描述、可选职业
  │  [城防署] [深渊议会] [自由行者联盟] [暮光商会]
  ▼
STEP 2: 选择职业 ──→ 按势力过滤可选职业，预览职业说明
  │  例: 城防署 → [外勤特工] [灵能调查员]
  ▼
STEP 3: 选择种族 ──→ 按势力+性别限制过滤
  │  禁选势力种族置灰不可选
  │  传说种族（天使/恶魔/纯机械生命体）标注 ⭐传说
  ▼
STEP 4: 选择性别 ──→ 按种族限制过滤
  │  锁性别种族自动设置性别下拉框并禁用（血族/天使→女，恶魔→男）
  ▼
STEP 5: 填写姓名/年龄 ──→ 实时预览区渲染角色概览
  │  预览内容：姓名·种族·年龄·职业·势力·种族特性摘要
  ▼
STEP 6: 确认创建 ──→ 写入所有初始变量（<UpdateVariable>）
  └── 路由至对应开场白
```

**联动规则**：
- 选定势力 → 职业下拉框重新渲染 → 禁选种族置灰
- 选定种族 → 锁性别自动设置 → 若传说种族锁定特殊开场路由
- 切换势力 → 若当前种族被新势力禁止，自动回退到人类
- 切换种族 → 若当前性别与新种族锁性别冲突，自动切换性别

## 七、任务自动清理机制

### 7.1 正则脚本

当 AI 输出中出现 `{{任务名}}完成` 或 `{{任务名}}放弃` 时自动清空任务变量：

```javascript
// 正则匹配: {{任意}}完成 或 {{任意}}放弃
const pattern = /\{\{([^}]+)\}\}(完成|放弃)/g;

// 匹配后执行：
// 1. 设置 任务.任务状态 = '已完成' or '已放弃'
// 2. 清空 任务.当前任务
// 3. 清空 任务.任务目标
// 4. 清空 任务.子任务
// 5. 保留 任务.已获取线索（历史记录）
```

### 7.2 正则脚本注册（pack 时自动注入）

| 脚本名 | 正则 | promptOnly | 动作 |
|--------|------|-----------|------|
| 任务完成检测 | `/\{\{([^}]+)\}\}(完成\|放弃)/g` | true | 触发变量清理 |

## 十、关键原则

| 原则 | 说明 |
|------|------|
| **单一数据源** | schema.ts 定义一切，initvar/规则从它派生 |
| **编辑源唯一** | 面板只改 `正则/`，条目只改 YAML |
| **打包唯一** | 只用 forge CLI，禁止其他脚本 |
| **后处理必跑** | 跳过导致序列化错误，MVU 失效 |
| **禁止编造** | 所有设定须有本计划文档支撑 |
| **变量白名单** | AI 只能写入规则中声明的路径 |
| **变更全局同步** | 改面板 → 同改 schema/initvar/规则 |
| **有 @@if 条目时跳过 configure** | configure 会将 constant 条目优化为 path，吞掉 @@if |
